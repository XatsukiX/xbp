LPとは

現代のWebマーケティングはランディングページが中心と言っても過言ではない時代になっています。 ランディングページとは広告をクリックした際にとび先となるページのこと。 縦長1ページに作られたセールス・コンバージョンに特化したページのことを指す。スマートフォンに。対応していることが多い 名前は離陸する際のランディングから来ている。


反応率

反応率＝CV率（CVR）
=コンバージョン数÷アクセス数

CVRをあげるにはLPを改善するのが効率が良い


LPの型

3つのnot/Read・Believe・Act。あくまでインターネットの情報なのでユーザーはあまり見ない。これらを促進する仕組み作りが必要。

以下はLPの基本形
  1. キャッチコピー
  2. 呼びかけ・導入
  3. 商品の魅力
  4. 根拠
  5. 共感
  6. 信頼・思い
  7. クロージング
