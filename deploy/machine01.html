How to use

3Dプリンターのつかいかた
fablabみなとみらいで使っている機種 Creality Ender-3 S1 Pro
使うソフト
  • Fusion360(モデリングソフト)
    • →モデリングソフトでなくても3Dデータが用意できればPrintableなどのサイトでも可
  • Ultimaker Cura(スライサーソフト)
Curaの日本語設定
  1. Curaを起動
  2. 上部のプレファランスからCuraを構成するへ
  3. 一般からlanguageを日本語へ 通貨を￥へ変更する
  4. ×を押した後Curaを落とす
  5. もう一度起動して日本語になればOK
Curaの初期設定
  1. 左上のプリンター一覧からプリンターの追加
  2. 非Ultimakerを選択後、非ネットワークプリンターを追加をクリック
  3. Creality3DのタブかたCreality Ender-3 S1 Proにチェックを入れて追加
  4. 次をおしてプリンターの設定は終わり
  5. プリンター一覧のよこのタブをクリック
  6. フィラメントからeSun→PLA→PLA+whiteを選択
詳細設定 プロファイルはLow→Standard→Dynamic→Superの順でクオリティが上がっていくが比例して時間がかかっていく
Lowは形をすぐに作りたい人向け、Superはクオリティを高めに作りたい人向け
サポートはサポート開始にチェックを入れるとサポートが生成されるが外すと生成されなくなる。
3Dプリンターの使い方

PC作業

  1. PCに3DプリンターについているSDカードを差しておく
  2. 作りたいデータのStlファイルを用意してCuraを起動する
  3. 左上のファイルアイコンからStlファイルを開く
  4. 詳細設定をしてスライスを押す
  5. プレビューを確認して大丈夫そうなら保存してSDカードを抜く。

    6. 注意！ 変換したGcodeデータの名前は英数字で構成するようにしてください!

本体作業

  1. データが入ったSDカードを本体にさして電源を入れる
  2. タッチパネルを操作し、印刷から該当データを選択する
  3. 再生ボタンを押して完成
