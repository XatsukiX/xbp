Yahooニュースを持ってくるプログラム

プログラムコード

  1. import requests
  2. from bs4 import BeautifulSoup
  3. import urllib
  4. import codecs
  6. #yahooから情報を持ってきます
  7. url = "https://www.yahoo.co.jp/l"
  8. html = requests.get(url)
  9. soup = BeautifulSoup(html.content,"html.parser")
  11. #filenameを定義しテキストファイルとして書き込みます
  12. filename = "linklist.text"
  13. with open(filename,"w", encoding="utf-8") as f:
  15. #中身。_2DunygeBZHdgHX_Gih3GC4ていうクラスのリンクとテキストを持ってくる指示。
  16.     topic = soup.find(class_="_2DunygeBZHdgHX_Gih3GC4")
  17.     for element in topic.find_all("a"):
  18.         print(element.text)
  19.         url2 = element.get("href")
  20.         link_url = urllib.parse.urljoin(url,url2)
  21.         f.write(element.text+"\n")
  22.         f.write(link_url+"\n")
  23.         f.write("\n")

プログラムの意図について

Yahooのサイトから今日のニュースを引っ張ってくるプログラムです。テキストファイルに見出しとURLがまとめられます。YahooのURLさえあれば開く必要がありません。
