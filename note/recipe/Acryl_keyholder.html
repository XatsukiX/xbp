アクリルキーホルダーのつくり方


加工方法

  • レーザー加工機
  • UVプリンター

使用するアプリケーション

  • Adobe illustrator

つくり方

  1. illustratorで画像を配置する
    2. （最大サイズが40～50㎜だとよい）
  2. 配置した画像を複製して片方を画像トレースでシルエットにする。
  3. 作成したシルエットを複製して塗りと線を反転させたうえでパスのオフセットを3㎜で行い、最後にアピアランスを分割する。
    4. （スタンドの場合は台座のデータも作る）
  4. それぞれをレイヤー分けする。
  5. 透明枠を作成
※レーザー用のデータは枠のデータを流用する。

レイヤーは以下のように構成すると良い。

  1. 透明枠
  2. 枠
  3. デザイン
  4. ホワイト
